# smt_lscustom

A highly optimized and feature-rich vehicle modification script based on the classic LS Customs, but enhanced for modern ESX servers with a clean UI and deep customization options.

## Features

- **Deep Customization**: Support for all GTA V vehicle modifications, including engine, brakes, transmission, suspension, armor, turbo, cosmetics, and more.
- **Advanced Paint Options**: Primary, secondary, and pearlescent colors with support for metallic, matte, and chrome finishes.
- **Wheels & Neons**: Extensive wheel categories (Sport, Muscle, Lowrider, SUV, etc.) and fully customizable neon lighting.
- **Job Integration**: Can be configured to be restricted to specific mechanic jobs.
- **Dynamic Pricing**: Prices can be scaled based on the vehicle's base value and the complexity of the upgrade.
- **Localized UI**: Support for multiple languages via `locales/`.

## Requirements

- [es_extended](https://github.com/esx-framework/es_extended)

## Installation

1. Copy `smtrp_lscustom` to your resources.
2. Configure your zones and job restrictions in `config.lua`.
3. Add `ensure smtrp_lscustom` to your `server.cfg`.
